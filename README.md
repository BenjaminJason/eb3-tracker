# EB-3 移民信息追踪（非官方）

> 信息科普用途，**不构成法律建议，不提供个案咨询，不推荐任何中介或律师**。所有数据以美国国务院及 USCIS 官方公布为准。

**Live:** https://benjaminjason.github.io/eb3-tracker/

## 功能
- EB-3/EB-2 办理流程时间线（含材料、时间窗口、耗时）
- Visa Bulletin 排期追踪：EB-2/EB-3 × 中国/印度/其它地区，近 24 月趋势
- 优先日期对照器（只陈述事实，不预测个案）
- 术语词典、材料填写注意事项、官方资源导航
- AI 信息助手（严格边界，不做个案判断）· 深浅色模式

## 数据更新
排期数据在 `data.js`，每月更新方法见文件顶部注释。
